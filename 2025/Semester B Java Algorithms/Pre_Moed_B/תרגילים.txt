תרגיל
א) 	כתבו קטע קוד הקולט מהמשתמש תו (char) אל תוך משתנה.
	התוכנית תדפיס את התו למסך.
ב) 	כתבו קטע קוד הקולט מהמשתמש תו (char) אל תוך משתנה.
	התוכנית תדפיס את ערכו המספרי המתאים (עפ"י טבלת ASCII).
ג) 	כתבו קטע קוד הקולט מהמשתמש מספר (int) אל תוך משתנה.
	התוכנית תדפיס את המספר למסך.
ד) 	כתבו קטע קוד הקולט מהמשתמש מחרוזת (String) אל תוך משתנה.
	התוכנית תדפיס את המחרוזת למסך.

תרגיל
כתבו קטע קוד הקולט מספר שלם חיובי (int) מהמשתמש, בטווח 0-127 (כולל).
התוכנית תדפיס את סימן ה-char המתאים למספר (עפ"י טבלת ASCII).
* ניתן להניח שהקלט תקין.

תרגיל - swap
כתבו קטע קוד, אשר מקבל שני משתנים a, b מטיפוס int.
הקוד יחליף ערכים בין a ו-b.
לדוגמא: a=3, b=5 ולאחר ההחלפה a=5, b=3

תרגיל - swap (אתגר)
כתבו אלגוריתם swap בין שני מספרים מטיפוס int,
ללא שימוש במשתני עזר!

תרגיל 3
כתבו פונ' אשר מקבלת מספר שלם חיובי בשם digit, מטיפוס int, בטווח 0-9.
הפונ' תדפיס למסך מחרוזת עם שם הספרה שהתקבלה.
Zero, One, Two, Three, Four, Five, Six, Seven, Eight, Nine
אם המספר לא בטווח, יוחזר null.

תרגיל 4
כתבו פונ' אשר מקבלת מחרוזת בשם str, המציינת שם של ספרה באנגלית.
Zero, One, Two, Three, Four, Five, Six, Seven, Eight, Nine
הפונ' תחזיר ערך מטיפוס int של הספרה המתאימה.
אם המחרוזת לא תקינה, יוחזר -1.

תרגיל 8
כתבו פונ' המייצרת תו (char) אקראי, ומדפיסה אותו למסך.
כלומר, הפונ' תדפיס למסך את אחד מהסימנים המוגדרים בטבלת ה-ASCII.
* רמז: אין פונ' מובנית לייצור תוים אקראיים. עלינו להבין איך תוים (char) מיוצגים במחשב,
לייצר ערך אקראי בהתאם, ולהמיר אותו לתו.

תרגיל 9
כתבו פונ' המייצרת ומחזירה אות לועזית (אנגלית) אקראית. האות יכולה להיות קטנה או גדולה.
כלומר, הבחירה בין אות גדולה לקטנה היא אקראית, וגם האות שנקבל נבחרה אקראית.
* רמז: אין פונ' מובנית לייצור תוים אקראיים. עלינו להבין איך תוים (char) מיוצגים במחשב,
לייצר ערך אקראי בהתאם, ולהמיר אותו לתו.

תרגיל 10
כתבו פונ' המייצרת ומחזירה תו (char) אקראי, מבין האפשרויות אות-קטנה, או אות-גדולה או ספרה-מספרית.
* רמז: אין פונ' מובנית לייצור תוים אקראיים. עלינו להבין איך תוים (char) מיוצגים במחשב,
לייצר ערך אקראי בהתאם, ולהמיר אותו לתו.

תרגיל - שש בש
כתבו קטע קוד המייצר 2 מספרים שלמים ואקראיים (רנדומלים), בטווח 1-6 (כולל).
התוכנית תדפיס למסך בשורה אחת את הערכים שיצאו.
בנוסף: 	אם יצאו בקוביות מספרים שווים - התוכנית תדפיס למסך "DOUBLE!"
		אם יצאו בקוביות המספרים 5, 6 - התוכנית תדפיס למסך "SHESH BESH!"

תרגיל - משחקי קלפים
א) כתבו קטע קוד המגריל מספר של קלף מחפיסת הקלפים, ומדפיס אותו למסך.
	טווח המס' של חפיסה הוא 2-10 (כולל)
ב) כתבו קטע קוד המגריל ערך של קלף מתוך חפיסת קלפים, ומדפיס למסך את הקלף שיצא.
	אם יצא קלף מספרי- יש להדפיס את מספרו למסך,
	אם יצא קלף עם שם (אס, נסיך, מלכה, מלך)- יש להדפיס את שמו למסך.
ג) כתבו פונ' המייצרת מחרוזת רנדומלית, המתארת קלף משחק - ומחזירה אותה.
המחרוזת תיהיה מורכבת מ: מס'/שם הקלף + הסימן שלו (עלה, לב, יהלום, תלתן).
* דגשים:
	(!!) ניתן לסמן קלפים מיוחדים במס' במקום בשמם.
	מס'/שם הקלף יוגרל אקראית.
	סימן הקלף יוגרל אקראית.
	ניתן להשתמש בתוים מיוחדים מתוך טבלת ה-ASCII המורחבת:
		char spade = '\u2660';	// עלה
	        char heart = '\u2665';	// לב
	        char diamond = '\u2666	';	// יהלום
	        char club = '\u2663';	// תלתן
לדוגמא:
3♠
Queen♦
8♥
Ace♦
ד) כתבו פונ' המקבלת מחרוזת כפרמטר, המתארת קלף-משחק (בהתאם לסעיף הקודם).
הפונ' תחזיר ערך int המייצג את חוזק הקלף.
כלומר מס' המייצג את הכח של קלף מבחינת מקומו הסידורי.
ניתן להניח שהקלפים נסיך, מלכה, מלך, אס מקבלים את הערכים 11,12,13,14 בהתאמה.
* ניתן להניח שהקלט תקין.
* רמז: 	ניתן להשתמש בפונ' המובנות charAt ו-startsWith של מחרוזת.
		ניתן להמיר מחרוזת מספרית ל-int בעזרת מניפולציה על char,
		או הפונ' המובנת Integer.valueOf שממירה מחרוזת ל-int.
ה) משחק-מלחמה
כתבו פונ' המדמה סיבוב יחיד במשחק מלחמה.
	המשחק: כל מתמודד מניח קלף רנדומלי שקיבל, והקלף עם הערך המספרי הגדול ביותר -
	מנצח!
הפונ' תרגיל 2 קלפים רנדומלים שביל 2 שחקנים,
לאחר מכן הפונ' תדפיס למסך את הקלפים שהוגרלו ומי השחקן המנצח (ניצחון אחד הקלפים, או תיקו).
