Fetch API Examples

Info:

Here, we demonstrate the use of Fetch API in Javascript.

GET

We use GET request to obtain Json data (textual) and Binary data (media).
  • For json data- use Response.json() method.
  • For binary data- use Response.blob() method.

POST

We use POST request to send data to the server,
e.g. saving data in database or pass crucial data to the server.
To make a POST request, fetch() must receive two parameters, in the following order:
  1. Endpoint (target) URL
  2. Object of request-options, like this-

 {
   method: some text..,
   headers: JS object holding metadata,
   body: JS object holding actual data
 }
