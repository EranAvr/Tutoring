Media Queries Examples

Intro

Media-queries are CSS tags, introduced in CSS-2.
Media-queries are a way to check a device capabilities and apply CSS code accordingly.
This way, we can group CSS attributes in different groups,
and activate different groups for different environments.

Cases

There are 2 ways we can apply media-queries:

On files

We can link our Html file to several css files with link tag, and using the media attribute we specify on which
device/environment we load this file.

On CSS property

We can use media-queries inside CSS code,
and create groups of CSS properties.
Each group will/won't be applied, according to its media-tag.
