Transforms Examples with CSS

'transform' is a CSS property, that takes as input one of the following functions:

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()
Each function has its parameters.

Examples:

transform: rotate(50px, 20px)
Sets element's upper-left corner to the specified position.

transform: rotate(20deg)

scale

transform: scale(0.5, 0.25)
Scales the element's width by 0.5 times,
and the element's height by 0.25 times.
TRANSFORMED
transform: scaleX(2)
Scales the element's width (X axis) by 2 times
TRANSFORMED
transform: scaleY(2)
Scales the element's height (Y axis) by 2 times
TRANSFORMED

Skew

transform: skew(20deg, 50deg)
Skews the element on its X-axis by 20deg,
and the element on its Y-axis by 50deg.
TRANSFORMED
transform: skewX(20deg)
Skews the element on its X-axis by 20deg
TRANSFORMED
transform: skewY(50deg)
Skews the element on its Y-axis by 50deg
TRANSFORMED

Matrix

transform: matrix(1, -0.3, 0, 1, 0, 0)
matrix combines all of the transforms into 1 function.
Parameters are as follow: matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()) .
Example is presented with margin top/left of 100px, for convenience.
