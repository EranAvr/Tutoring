Transitions and Animations Examples with CSS

Transitions

Transitions are used to change CSS properties smoothly.
We add 'transition' property to the desired element,
and specify for what property we activate the transition, and for what duration.

We can define transition with the following prepoerties:

  1. transition-property: ;
  2. transition-duration: ;
  3. transition-timing-function: ;
  4. transition-delay: ;
Or to use the shorthand property transition,
that takes the whole 4 properties altogether.

transition: color 2s ease-in-out 1s;

Animations

Animations are used to create a 'slideshow' like effect,
for changing CSS properties smoothly.
The animation system uses a tag called '@keyframes' for creating
different iterations/states for our animation,
and a set of CSS animation-properties to determine the behaviour.

We can define animations with the following prepoerties:

  1. animation-name: ;
  2. animation-duration: ;
  3. animation-timing-function: ;
  4. animation-delay: ;
  5. animation-iteration-count: ;
  6. animation-direction: ;
Or to use the shorthand property animation,
that takes the whole 6 properties altogether.
